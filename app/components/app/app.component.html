Setup

Import Mapbox GL

The first step is to import the Mapbox GL library. You can do it how ever you like but if you want to get up and running quickly paste the follwoing code into your head tag in index.html


			<script src='https://api.mapbox.com/mapbox-gl-js/v0.18.0/mapbox-gl.js'></script>
			<link href='https://api.mapbox.com/mapbox-gl-js/v0.18.0/mapbox-gl.css' rel='stylesheet' />

Access Token

Before you start playing with some maps you have to tell mapboxgl to use your accessToken

Note! If you do not have an access token then you need to head over to mapbox.com, create an account and get your own token from your account page. 

			
		//Switch out this fake accessToken against yours
		mapboxgl.accessToken = "pk.ertv345avewrva4bnNzb26haSIsImEiOiJIVVByOWJnIn0.Dq2xmRfgss5sergserga4ta23";

Components

<mapbox-gl-map>

This is the root component, you will change position, style, transition parameters as well as hooking up to the map objects different events using this component.

Styling


			<mapbox-gl-map
				[style]="Mapbox styleURL String (can be found on your mapbox developer page)">
			</mapbox-gl-map>

You can use Mapbox style urls to make your map look pretty.

Language


			<mapbox-gl-map
				[language]="Language Id String">
			</mapbox-gl-map>

If you want to change the maps language you can do so by modifying the language property.

Supported Language IDs:

  • fr (French)
  • ru (Russian)
  • de (German)
  • es (Spanish)

Positioning


			<mapbox-gl-map
				[center]="accepts an array like this [lat,lng]"
				[bearing]="Number between 0 to 360"
				[pitch]="Number between -180 to 180"
				[zoom]="Number between 0 to 22">
			</mapbox-gl-map>

These values control the cameras position. Mapbox gl supports zooming, tilting(pitch), rotating(bearing) as well as setting the cameras position in longitude and latitude.

By default the when these properties change the map will automatically reposition the camera to fit your constraints.

Animation


			<mapbox-gl-map
				[animate]="Bool, tells mapbox if it should animate between positions."
				[duration]="Number, the duration of the transition animation in seconds"
				[speed]="Number, the speed of the animation default 1.2"
				[curve]="Number, the amount of zooming that will happen during the animation">
			</mapbox-gl-map>

These parameters tell mapbox how changes in the position properties should be animated. The curve and speed props can be a little harder to understand so to read more click here
